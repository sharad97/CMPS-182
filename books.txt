City Books:

The following is an Books schema for City Books store database. There are three tables:

books, authors, and customers. Each table has a primary key.


books ( book_id, title, author_id)
Primary key: ( book_id )

authors ( author_id, first_name, last_name)
Primary key: ( author_id )

customers ( customer_id, first_name, last_name, phone_number )
Primary key: ( customer_id)
